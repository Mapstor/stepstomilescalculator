### Key Formulas
- Miles = (Steps × Stride Length ft) ÷ 5,280
- Walking stride ≈ Height × 0.415
- Running stride ≈ Height × 0.52
- Avg walking: ~2,100 steps/mile
- Avg running: ~1,600 steps/mile
